# Executar local

## Docker RUN - Desenvolvimento

### Windows
docker run --rm -it --name api-flask-csv-container -e TZ=America/Sao_Paulo -p 8080:5000 -p 5678:5678 -v .:/app -v .\data:/app/data -v .\media:/app/src/media -w /app/src --entrypoint /app/scripts/entrypoint.sh python:3.12-slim bash

### Linux
docker run --rm -it --name api-flask-csv-container -e TZ=America/Sao_Paulo -p 8080:5000 -p 5678:5678 -v .:/app -v ./data:/app/data -v ./media:/app/src/media -w /app/src --entrypoint /app/scripts/entrypoint.sh python:3.12-slim bash

### Ambos
docker exec -it api-flask-csv-container bash

## Docker Compose

### Ambos - Produção
docker compose -f container/docker-compose.yml build
docker compose -f container/docker-compose.yml up -d

### Ambos - Desenvolvimento
docker compose -f container/docker-compose-dev.yml up -d
docker logs --tail 100 -f api-flask-csv-container
docker exec -it api-flask-csv-container bash
